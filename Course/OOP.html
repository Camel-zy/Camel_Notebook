    Template¶

    类模板的参数错误的说法是： B A.可以有多个 B.可以有0个 C.不能有基本数据类型 D.参数不能给初值

    静态成员¶

    模板类中静态成员的特殊之处：每种类型的实例的静态成员相互独立，不共享。例如：

    template <typename T>
void test(const T&x) {
    static int count = 0;
    cout << "x = " << x << " count = " << count << endl;
    ++count;
    return;
}
 
void main() {
    test<int>(2);
    test<int>(2);
    test<double>(2.2);
}
// output:
x = 2 count = 0
x = 2 count = 1
x = 2.2 count = 0

    类模板显式具体化¶

    类模板有两种特化：全特化和偏特化。

    template<class T1,class T2>
class MyClass {…};

template<>
class MyClass<int,int>{…};      //全特化

template<class T1>
class MyClass<T1,int>{…};       //偏特化
